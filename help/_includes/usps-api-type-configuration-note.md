# Opções de configuração do tipo de API USPS

>[!NOTE]
>
>Essas instruções incluem etapas para selecionar o tipo de API USPS para integração: API de ferramentas da Web ou APIs REST. As opções de configuração do tipo de API só estarão disponíveis se você tiver aplicado o [patch de qualidade de migração da API REST do USPS](https://experienceleague.adobe.com/docs/commerce-knowledge-base/kb/troubleshooting/known-issues-patches-attached/usps-rest-api-migration-patch.html)&#x200B;(AC-1520) ao seu aplicativo do Commerce. Se você não tiver aplicado o patch, o seletor de tipo de API não estará disponível e a API de ferramentas da Web do USPS será usada por padrão.<br>As APIs do USPS são o método preferido para integração com o USPS. A API de ferramentas da Web do USPS está obsoleta e pode ser removida em versões futuras.
